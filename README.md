# Personal Journal Manager (Python)

A simple command-line journal application written in Python.
It lets you write, view, search, and delete personal journal entries stored in a text file.

## Features

- Add new journal entries with date & time
- View all saved entries
- Search entries by keyword or date
- Delete all journal entries
- Exit safely from the program

## How to Run

- Run the program:
  
  - python journal.py
